🚀 Ligraphi Documentation

Ligraphi is a powerful interactive graph builder that allows you to create, edit, and visualize line charts using both UI interactions and a custom code syntax.

🔥 Features

  • Interactive line graph builder
  • Click to add points
  • Drag to edit values
  • Edit labels and categories
  • Multiple datasets
  • Sparkline mode
  • Code-based graph creation
  • Export as PNG

💻 Syntax Guide

Ligraphi uses a simple custom syntax to define graph data.

add point = [ label = {A} category = {January} value = {100} ]

Fields

  • label → internal identifier
  • category → X-axis label
  • value → Y-axis value

📊 Example

add point = [ label = {Jan} category = {January 2025} value = {120} ] add point = [ label = {Feb} category = {February 2025} value = {140} ]

This will generate a line graph with two data points.

⚡ Advanced Usage

Multiple Points

add point = [ label = {A} category = {Jan} value = {50} ] add point = [ label = {B} category = {Feb} value = {80} ] add point = [ label = {C} category = {Mar} value = {120} ]

Tips

  • Order matters (defines graph sequence)
  • Values should be numeric
  • Categories define X-axis labels
